初回のみ EmailJS の設定 が必要です（約5分）

EmailJS 初期設定
1
EmailJS に無料登録する
emailjs.com を開いて「Sign Up Free」でアカウント作成。Googleアカウントでも可。
2
メールサービスを追加する
左メニュー「Email Services」→「Add New Service」→ Gmail を連携。完了後の Service ID をコピー。
3
テンプレートを作成する
「Email Templates」→「Create New Template」
• Subject 欄に {{subject}}
• Message 欄に {{message}}
• To Email 欄に {{to_email}}
と入力して保存。Template ID をコピー。
4
Public Key を取得する
「Account」→「General」タブ→「Public Key」をコピー。
